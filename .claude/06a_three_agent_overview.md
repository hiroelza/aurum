# 3エージェント評価システム実行手順

このファイルは、投資判断における3エージェント評価システムの実行方法を定義します。

---

## 概要

**3エージェント評価システム**は、投資判断を3つの視点から評価し、バランスの取れた意思決定を支援するシステムです。

### 3つの視点

1. **hayato（investment-advisor-hayato）** - 伊藤ハヤト投資哲学評価者
   - 視点: 投資哲学との整合性
   - 評価基準: 7つの原則（低コスト、分散、パッシブ等）

2. **researcher（investment-researcher）** - 統計データ分析者
   - 視点: 統計データ・過去実績に基づく分析
   - 評価基準: モンテカルロシミュレーション、家計収支分析等

3. **japanese（investment-advisor-japanese）** - 日本税制・ライフプラン専門家
   - 視点: 日本の税制・ライフプラン適合性
   - 評価基準: NISA/iDeCo活用、ライフイベント考慮等

---

## いつ使用するか

### 必須のケース（Level 4タスク）

以下の場合は**必ず3エージェント評価を実施**してください：

- 🔴 新規投資商品の購入判断
- 🔴 ポートフォリオの大幅な変更
- 🔴 投資方針の変更
- 🔴 NISA/iDeCo枠の使い方の変更
- 🔴 高額な投資（100万円以上）

### 推奨のケース

以下の場合も評価を推奨：

- 月次投資額の変更
- リバランスの実施
- 自社株売却のタイミング判断

### 不要なケース

以下は評価不要：

- 情報収集・市場調査のみ
- 用語の説明
- 過去データの確認

## 実行手順

### Step 1: 事前準備

#### 1-1. INVESTMENT_PROFILE.mdの確認

```bash
# 最新の資産状況、投資判断履歴を確認
cat ./INVESTMENT_PROFILE.md
```

#### 1-2. 必要情報の整理

評価に必要な情報をまとめる：
- 投資商品名・詳細
- 投資額
- 期待リターン・リスク
- 費用（経費率等）
- 投資期間

---

### Step 2: 3エージェントの起動

#### 実行方法A: 並行実行（推奨）

3エージェントを同時に起動し、独立した評価を得る：

```markdown
以下の投資判断について、3エージェント評価を実施してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 目的: {目的}
- 期間: {期間}

【評価依頼】
investment-advisor-hayato、investment-researcher、investment-advisor-japaneseの
3エージェントで並行評価し、それぞれのスコア（0-100点）と支持度（0-100%）を提示してください。

【追加質問】⭐NEW
各エージェントは以下の質問にも回答してください：
1. 「私が見落としていることは何？」
   - リスクの過小評価、前提条件の誤認、代替案の見逃しはないか
2. 「もしあなたが私の立場ならどうする？」
   - エージェントの立場での最適解と優先順位の再評価
3. 「他に知っておくべきことはありますか？」
   - 追加の注意事項、市場環境の変化、過去の類似ケースからの教訓
```

#### 実行方法B: 順次実行

複雑な判断の場合、順次実行して前のエージェントの結果を踏まえることも可能：

```markdown
1. まずinvestment-advisor-hayatoで哲学的評価を実施
2. 次にinvestment-researcherで統計分析を実施（hayatoの結果を踏まえて）
3. 最後にinvestment-advisor-japaneseで税制・ライフプラン評価を実施
```

---

### Step 3: 評価結果の記録

#### 評価フォーマット

各エージェントは以下の形式で評価を提供：

```markdown
## {エージェント名}の評価

### 総合評価
- **スコア**: XX/100点
- **支持度**: XX%
- **判定**: [強く支持 / 条件付き支持 / 中立 / 懸念あり / 強く反対]

### 評価根拠
{具体的な理由}

### 推奨事項
- {推奨1}
- {推奨2}

### 追加質問への回答 ⭐NEW
#### 見落としている点
{リスクの過小評価、前提条件の誤認、代替案の見逃し等}

#### 私が同じ立場なら
{エージェントの立場での最適解と優先順位}

#### 他に知っておくべきこと
{追加の注意事項、市場環境の変化、過去の教訓等}
```

#### 統合評価シート

3エージェントの評価を統合：

```markdown
## 3エージェント評価結果

| エージェント | スコア | 支持度 | 判定 |
|-------------|--------|--------|------|
| hayato      | XX/100 | XX%    | {判定} |
| researcher  | XX/100 | XX%    | {判定} |
| japanese    | XX/100 | XX%    | {判定} |
| **平均**     | **XX/100** | **XX%** | - |

### 総合判断
{3つの評価を総合した判断}
```

---

### Step 4: 意見相違時の統合ルール

#### パターン1: 全員が支持（スコア70点以上）

→ **実施を推奨**

#### パターン2: 2人が支持、1人が反対

→ **条件付きで実施可能**
- 反対理由を確認
- 反対エージェントの懸念に対処できるか検討
- 対処可能なら実施、不可能なら再検討

#### パターン3: 過半数が反対（2人以上がスコア50点未満）

→ **実施を見送り**
- 投資内容を見直す
- 代替案を検討

#### パターン4: hayatoが強く反対（スコア30点未満）

→ **原則として実施しない**
- 投資哲学に反する判断は長期的にリスク
- どうしても実施したい場合は、哲学との矛盾を明示的に記録

#### スコアリングの重み付け

**基本**: 3エージェント均等（各33.3%）

**調整**（状況に応じて）:
- 長期投資（10年以上）: hayato 40%、他30%ずつ
- 短期戦術（5年未満）: researcher 40%、他30%ずつ
- 税制メリット重視: japanese 40%、他30%ずつ


## 実行例

### 例1: 新規投資信託の購入判断

```markdown
【投資内容】
- 商品名: eMAXIS Slim 全世界株式（オール・カントリー）
- 投資額: NISA枠で月10万円
- 目的: 老後資金形成
- 期間: 19年間

【評価結果】
| エージェント | スコア | 支持度 | 判定 |
|-------------|--------|--------|------|
| hayato      | 95/100 | 95%    | 強く支持 |
| researcher  | 90/100 | 85%    | 強く支持 |
| japanese    | 92/100 | 90%    | 強く支持 |
| **平均**     | **92/100** | **90%** | - |

【総合判断】: 実施を強く推奨
- 全エージェントが高評価
- 投資哲学に完全合致
- 税制メリット最大化
```

### 例2: 個別株への投資検討（意見相違あり）

```markdown
【投資内容】
- 商品名: Apple株
- 投資額: 200万円
- 目的: ポートフォリオ分散
- 期間: 10年間

【評価結果】
| エージェント | スコア | 支持度 | 判定 |
|-------------|--------|--------|------|
| hayato      | 25/100 | 10%    | 強く反対 |
| researcher  | 65/100 | 60%    | 条件付き支持 |
| japanese    | 55/100 | 50%    | 中立 |
| **平均**     | **48/100** | **40%** | - |

【総合判断】: 実施を見送り
- hayatoが強く反対（個別株集中、パッシブ哲学に反する）
- 既に自社株でIT/ハイテク集中リスクあり
- セクター集中をさらに悪化させる

【代替案】: 全世界株式インデックスファンドで分散投資
```
