# 3エージェント評価システム - 詳細手順

このファイルは、3エージェント評価システムの詳細手順、トラブルシューティング、簡易評価モードを定義します。

**基本的な使い方は `06a_three_agent_overview.md` を参照してください。**

---

## トラブルシューティング

### Q1: エージェント間で大きく意見が分かれた場合は？

**A**: パターン2（2人支持、1人反対）の統合ルールに従ってください。反対理由を確認し、対処可能かを判断します。

### Q2: すべてのエージェントが「条件付き支持」の場合は？

**A**: 各エージェントが提示した条件をすべて満たせるか確認してください。満たせる場合は実施可能、満たせない場合は再検討。

### Q3: hayatoのスコアが低いが、他2つが高評価の場合は？

**A**: hayatoが30点未満の場合は原則見送り。30-50点の場合は、投資哲学との矛盾を明示的に記録した上で慎重に判断。

### Q4: 評価に時間がかかりすぎる場合は？

**A**: 簡易評価モードを使用してください。詳細は後述の「簡易評価モード」セクションを参照。

---

## 簡易評価モード

即座の判断が必要な場合や、軽微な変更の場合は、以下の簡易評価モードを使用できます。

### モード1: 単独エージェント評価（最速: 30秒-1分）

**使用タイミング**:
- 軽微な投資額の変更（月額5万円以下）
- 既存商品の追加購入（過去に3エージェント評価済み）
- 緊急性の高い判断

**実行方法**:

#### 1-A: hayatoのみ評価（哲学チェック）

```markdown
investment-advisor-hayatoで以下の判断を評価してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 目的: {目的}

伊藤ハヤト氏の7原則に照らして、スコア（0-100点）と支持度（0-100%）を提示してください。
```

**適用ケース**: 哲学との整合性が最重要な判断（例: 新しい投資商品の評価）

#### 1-B: researcherのみ評価（統計チェック）

```markdown
investment-researcherで以下の判断を評価してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 期待リターン: {リターン}
- リスク: {リスク}

過去の統計データに基づいて、スコア（0-100点）と支持度（0-100%）を提示してください。
```

**適用ケース**: 統計的妥当性が最重要な判断（例: リスク・リターンの確認）

#### 1-C: japaneseのみ評価（税制チェック）

```markdown
investment-advisor-japaneseで以下の判断を評価してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 使用枠: NISA成長投資枠 / つみたて投資枠 / iDeCo / 特定口座

日本の税制とライフプランに照らして、スコア（0-100点）と支持度（0-100%）を提示してください。
```

**適用ケース**: 税制最適化が最重要な判断（例: NISA枠の使い方）

---

### モード2: 2エージェント評価（推奨: 1-2分）

**使用タイミング**:
- 中程度の投資判断（5-50万円）
- 既存戦略の微修正
- 時間制約があるが慎重さも必要な場合

**実行方法**:

#### 2-A: hayato + researcher（哲学×統計）

```markdown
investment-advisor-hayatoとinvestment-researcherで以下を評価してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 目的: {目的}

【評価依頼】
- hayato: 哲学的評価（スコア・支持度）
- researcher: 統計的評価（スコア・支持度）

2つの平均スコアと総合判断を提示してください。
```

**適用ケース**: 長期投資判断で税制考慮が不要な場合

#### 2-B: hayato + japanese（哲学×税制）

```markdown
investment-advisor-hayatoとinvestment-advisor-japaneseで以下を評価してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 使用枠: {NISA/iDeCo/特定口座}

【評価依頼】
- hayato: 哲学的評価（スコア・支持度）
- japanese: 税制・ライフプラン評価（スコア・支持度）

2つの平均スコアと総合判断を提示してください。
```

**適用ケース**: NISA/iDeCo枠の使い方を検討する場合

#### 2-C: researcher + japanese（統計×税制）

```markdown
investment-researcherとinvestment-advisor-japaneseで以下を評価してください：

【投資内容】
- 商品名: {商品名}
- 投資額: {金額}
- 期待リターン: {リターン}

【評価依頼】
- researcher: 統計的評価（スコア・支持度）
- japanese: 税制・ライフプラン評価（スコア・支持度）

2つの平均スコアと総合判断を提示してください。
```

**適用ケース**: 既に哲学に合致していることが明らかな商品（例: 全世界株式インデックス）の金額調整

### モード3: 3エージェント評価（完全版: 3-5分）

**使用タイミング**:
- 高額な投資判断（50万円以上）
- 新規投資商品の購入
- 投資方針の変更
- ポートフォリオの大幅な変更

**実行方法**: 上記「実行手順 Step 2」を参照

---

### 簡易評価モードの使い分けフローチャート

```
投資判断が必要
    ↓
    ├─ 金額が5万円以下 → 単独エージェント評価（モード1）
    ├─ 金額が5-50万円 → 2エージェント評価（モード2）
    └─ 金額が50万円以上 → 3エージェント評価（モード3、完全版）

時間制約がある
    ↓
    ├─ 即座に判断必要 → 単独エージェント評価（モード1）
    ├─ 1-2分は使える → 2エージェント評価（モード2）
    └─ 3-5分は使える → 3エージェント評価（モード3）

判断の重要度
    ↓
    ├─ 軽微な変更 → 単独エージェント評価（モード1）
    ├─ 中程度の判断 → 2エージェント評価（モード2）
    └─ 重要な判断 → 3エージェント評価（モード3、必須）
```

---

### 簡易評価モードの注意点

1. **記録は必須**: 簡易評価でも必ずINVESTMENT_PROFILE.mdに記録する
2. **重要判断では使用禁止**: 50万円以上、新規商品、方針変更では必ず3エージェント評価
3. **定期的な完全評価**: 3ヶ月に1回は3エージェント評価で全体を見直す
4. **評価の偏り**: 単独エージェント評価では視点が偏るリスクを認識する

## 重要な原則

1. **教育費（Tier 1）を最優先**: 2,400万円達成まではハイリスク投資禁止
2. **投資哲学の一貫性**: hayatoの評価が低い場合は慎重に
3. **記録の徹底**: すべての判断をINVESTMENT_PROFILE.mdに記録
4. **感情を排除**: データと原則に基づく判断
