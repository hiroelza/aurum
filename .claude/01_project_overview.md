# プロジェクト概要・投資哲学

このファイルは、aurumプロジェクトの目的、投資哲学、重要な数値パラメータを定義します。

---

## 🔴 これらのルールを必ず守ること

### 投資判断
- `INVESTMENT_PROFILE.md` を **必ず読むこと**（投資判断前）
- 3エージェント評価（hayato、researcher、japanese）を **必ず実施すること**
- すべての判断を `INVESTMENT_PROFILE.md` に **必ず記録すること**（点数0-100、理由を含む）
- 教育資金を危険にさらすハイリスク投資は **絶対に禁止**（Tier 1: `INVESTMENT_PROFILE.md`参照）
- すべての投資機会よりも教育資金の安全性を **優先すること**

### コード実行
- Pythonスクリプトは `aurum/` ディレクトリから **必ず実行すること**
- サブディレクトリ（python/main/、python/analysis/等）からの実行は **禁止**
- すべてのファイルパスは `python/config.py` を **必ず使用すること**（IMAGE_DIR、CSV_DIR等）
- スクリプト実行前に `pwd` でカレントディレクトリを **必ず確認すること**

### データプライバシー・セキュリティ
- すべての金融データを **機密情報として必ず扱うこと**
- 共有ドキュメントに実名や具体的な金額を **絶対に含めないこと**
- Git コミット前に機密情報がないか **必ず確認すること**
- 個人金融データを含むファイルをコミットしようとする場合、ユーザーに **必ず警告すること**

### 作業プロセス
- 投資関連の変更は **Plan Mode を必ず使用すること**
- 複雑なタスク（Level 4）では4段階プロセス（要件定義→要件詳細→TODO→実行）を **必ず守ること**
- 簡単なタスク（Level 1-2）ではプロセスを省略しても良い（`.claude/03_work_process.md` に定義）
- すべての作業成果物を `outputs/reports/working/yyyymmdd/` に **適切なファイル名で必ず保存すること**

## プロジェクトの目的

このディレクトリは**家計・投資管理プロジェクト**です。個人の資産運用、教育費計画、老後資金シミュレーションを統計的に分析し、データに基づいた投資判断を支援します。

### 主要な機能
- 家計収支のバランス分析
- 60歳時点の資産予測シミュレーション
- 教育費達成確率のモンテカルロシミュレーション
- 投資判断の記録と評価（3エージェント評価システム）
- ライフプラン全体の管理

## 投資哲学と原則

### 採用している投資哲学

**伊藤ハヤト氏の投資哲学**を基本とする：

1. 低コスト・インデックス投資
2. 最大限の分散（全世界株式）
3. 市場タイミングを取らない
4. パッシブ運用のみ
5. 個別銘柄やセクター集中を避ける
6. 長期保有（バイ&ホールド）
7. "Time in the market > Timing the market"

### 3エージェント評価システム

投資判断は以下の3つの視点で評価（0-100点、支持度0-100%）：

1. **hayato（哲学）**: 伊藤ハヤト氏の投資哲学との整合性
2. **researcher（統計）**: 統計データ・過去実績に基づく分析
3. **japanese（税制・リスク）**: 日本の税制・ライフプラン適合性

**詳細な実行手順:**
- **`.claude/06a_three_agent_overview.md`** - 評価システムの概要
- **`.claude/06b_three_agent_detailed.md`** - 詳細な実行手順
- **`.claude/06c_weight_adjustment_rules.md`** - ウェイト調整ルール（学習目的、税制最適化など）
- **`.claude/09_baseline_evaluation.md`** - ベースライン評価の記録ルール（実投資せず学習記録）

#### 🔴 3エージェント評価時の追加質問（必須）⭐NEW

**重要**: 3エージェント評価を実施する際は、**ユーザーが明示的に依頼しなくても**、Claude Codeは自動的に以下の追加質問を含めて評価を実施すること：

1. **「私が見落としていることは何？」**
   - リスクの過小評価、前提条件の誤認、代替案の見逃しはないか

2. **「もしあなたが私の立場ならどうする？」**
   - エージェントの立場での最適解と優先順位の再評価

3. **「他に知っておくべきことはありますか？」**
   - 追加の注意事項、市場環境の変化、過去の類似ケースからの教訓

これらの質問により、投資判断の思考の深度を高め、見落としを防止する。

### 投資の優先順位

**Tier 1（最優先）: 教育費の確実な確保**

教育費の目標額は家族構成により変動します。**必ず `INVESTMENT_PROFILE.md` を参照してください。**

- **子供がいない場合**: Tier 1は不要、Tier 2（老後資金）が最優先となる
- **子供1人の場合**: 目標額 約1,200万円（大学進学時期は家族構成参照）
- **子供2人の場合**: 目標額 約2,400万円（大学進学時期は家族構成参照）
- **原則**: この目標達成まで、ハイリスク投資は禁止

**Tier 2（次優先）: 老後資金の形成**
- 目標: 退職時点での目標資産額（`INVESTMENT_PROFILE.md` 参照）
- 例: 60歳時点で3,000-5,000万円以上

**Tier 3（余剰資金）: 実験的投資**
- Tier 1達成後の余剰資金のみ
- 上限: 総資産の10-15%

#### 主要リスク要因

1. **米国関税政策**
   - 実効関税率の上昇→グローバル企業・米消費への悪影響
   - サプライチェーン見直しによる資本投資の地域間格差

2. **中国過剰生産**
   - 2025年にダブついた在庫→欧州・東南アジアへ悪影響
   - コモディティ価格の軟調

3. **構造的インフレ（米国）**
   - 国内製造回帰によるコスト上昇
   - FRBの利下げペース鈍化リスク

4. **財政リスク**
   - 主要国の財政赤字拡大
   - ソブリン債に財政プレミアム上乗せリスク

5. **システム金融リスク**
   - プライベート・アセット関連企業の株価下落
   - クレジット市場への警戒

#### 投資判断への影響

**株式**:
- 米国株式市場は最高値圏だが割高感
- 「価格」でなく「価値」の上昇が期待できる銘柄を厳選
- オーガニックな成長を持続する企業に注目

**債券**:
- 投資適格社債・証券化商品（高格付け）を選好
- 新興国は米ドル建て社債を優先
- ソブリン債は財政リスクに注意

**全般**:
- 過剰流動性による資産インフレに警戒
- セミマクロ・ミクロへの影響を注視
- 銘柄選別の重要性が増大
